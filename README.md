## Setup

### Proyecto

Instalar Dependencias

```bash
npm instal
# ó
npm i
# ó
docker compose up build
```

Correr proyecto

```bash
npm run dev
# ó
docker compose up -d
```

### Storybook

```bash
npm run storybook
```

### Jest

```bash
npm run test
# correr test especifico
npm run test Button.test
```
